Tank Exporter allows you to take pictures of the current screen.

After you load a tank or import an FBX, use Screen Capture under the Export/Extract menu item.
The zoom and position of the view effects the saved image.
Screen Capture only writes PNG files.

I have given you a nice selection of sizes to save your PNG and also any size you wish to use.
You should make these custom sizes even numbered. (Divisable by 2).
Also, there are presets for the World of Tanks image Icons and the Mods Portal cover.

Pick the background type and click Save.
The Image radio button will allow you to load a image as the background.
When set, it will ask for the image to load after you select where to save your capture.
For best results, the image should have the same proprotions of the image size you are capturing.

If you want to create an image for a forum, I recommend using color background and setting the color to match that of the forums.
The reason for this is anti-aliasing works with colors and not alpha/transparent backgrounds.
