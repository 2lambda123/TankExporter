Other useful tools you should have in your tool box.

Swizzler. A Must Have! A good tool for working with DDS and PNG files. I wrote this specifically for working with World of Tank texture files. LINK
WoT XML Editor. A Must have! This is a tool I wrote to convert and edit the compressed XML files the game uses. You can not edit the games XMLs in the format they are in. LINK
NVIDIA plugin. For Adobe Photoshop. A tool for creating Normal Maps. LINK
Gimp 2. Very useable tool for editing image files.. You can download a plugin for working with DDS files. LINK. Gimp 2 DDS and Normal Map plugin LINK
Paint.NET. Another useful and free painting program. LINK
3DS Max. Expensive but worth the money. LINK
FBX Review. A tool by Autodesk to view FBX files. LINK
