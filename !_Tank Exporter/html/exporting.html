Exporting a Tank....

To export a tank, first load one...

After, on the top menu, under Export/Extract, click on Export FBX.

Select a place to store your FBX. I usually create a work folder on my desktop called "Exported FBX".
After you chose where to write the FBX to, the FBX Exporter panel will popup. (purposely spelled wrong)

Blender Compatible FBX writes a FBX that works with Blender.
This format is a work around for Blenders inability to load instanced materials and mesh models.
Blender Compatible FBX files will be larger because they contain a NEW material and a NEW model for every Bend bone pin Marker.
Read more below....

Flip U and V can be handy if for some reason the texture mapping is wrong in your 3D editing application.
Convert Normal Maps. This converts the normal maps in to standard tangent maps.
Flip Y Normal. If bum[ mapping looks upside down, checking this should fix the problem.
Export Textures does what it says.. Leave this checked..
Tank Exporter will create a sub folder with the name of the tank and place all the textures within it.
Textures will never be over written when exporting by Tank Exporter.
This is to protect any changes you make to the PNGs Tank Exporter created.
Create Binary file.. This forces a binary file to be created. Its much smaller than ASCII but non-man-readable.
Leave this checked unless you want to try and edit a FBX by hand. (Not recommended).
Blender and Spilt Textures check boxes force the exporter to create a texture for each model.
Stand alone primitives FBX export mixes and creates custom texture maps.
Creating these textures is the only way to display the models textures correctly.
If UV coordinates cause overlapping textures, checking Split Textures fixes this by creating a new texture for every map.
This can cause a lot of wasted space on the maps but there is no other solution.
Do Not Export Markers. Stops the exporting of the Bend Bone markers.

Click Export to write out the new FBX file. After, you can import it in your 3D application.

NOTE: Blender FBX files exported from Tank Exporter can be imported in to other 3D apps such as 3DS Max.
The difference is, there will be a lot of materials. One for each Bend bone pin Marker.


The current FBX plugin for Blender can NOT import the version of FBX Tank Exporter creates.
You will need to download Autodesk's free FBX Converter. Use it to convert Tank Exporter's FBX to version 2013.

Another very serious issue with Blender's FBX plugin is, when importing the materials, by default, the opacity is set to ZERO.
This does not affect Tank Exporter but other apps will show the materials as transparent.
FBX Converter's FBX viewer may not show the models! FBX Preview may have issues as well
You need to fix the Alpha in blender 2.79.
In Blender, You can disable transparency in the windows material setting or edit the materials opacity.
You can change the ALPHA to 1.0 for each material for the tanks components.
Blender 2.8 does not have this problem.
